# kotlin 协变 和 逆变



|            |                             协变                             |                             逆变                             |                   不变                   |
| ---------- | :----------------------------------------------------------: | :----------------------------------------------------------: | :--------------------------------------: |
| **Kotlin** |   实现方式： **<out T>** ,只能作为消费者，只能读取不能添加   |   实现方式：**<in T>** ,只能作为生产者，只能添加，读取受限   | 实现方式：**<T>** 既可以添加，也可以读取 |
| **Java**   | 实现方式： **<? extends T>** , 只能作为消费者，只能读取不能添加 | 实现方式： **<? super T>** , 只能作为生产者，只能添加，读取受限 | 实现方式：**<T>** 既可以添加，也可以读取 |
